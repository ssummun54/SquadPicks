SquadPicks is a free 2026 FIFA World Cup prediction game. No real money involved.

**Domain:** squadpicks.co (owned, connecting to Vercel now)

**Stack:** Next.js 16 (App Router) + TypeScript + Tailwind v4 + Supabase (Auth + PostgreSQL) + Vercel

**Auth:** Email+password with OTP verification, Google OAuth (pending Google Cloud Console setup)

**Current milestone:** Deploying to squadpicks.co for beta testing

**Up next after beta deploy:** Make SquadPicks a PWA (manifest.json, service worker, push notifications for scores + prediction reminders)

**Why:** Built as a free prediction game for the 2026 World Cup.

**How to apply:** When suggesting infra, URLs, email addresses, or deployment config, use squadpicks.co as the domain.
